Grid and Flexbox
  • Grid is two-Dimentional but Flexbox is One Dimensional
  • Grid can flex combination of items through space-occupying Features, on the otherhand flexbox can push content element to extreme alignment
  • Grid is Layout First support type, but flex-box is content First

Bootstrap vs Tailwind CSS Both frameworks are open-source and are designed for utility and responsive mobile web development. Bootstrap provides HTML, CSS, and JavaScript design templates for a variety of interface components. They also use the same set of tools for building websites. To make your design process easier, Tailwind CSS has a utility-first framework. You can use it like regular CSS, but it includes a number of new features.
both frameworks offer a lot of similarity, the main difference is the degree of customization you can do with them. Tailwind is much more flexible and allows you to add prefixes and utility classes to your CSS files, while Bootstrap is not flexible enough. Here are the pros and cons of each framework.

CSS Box Model: The CSS box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages. It can be used as a toolkit for customizing the layout of different elements. The web browser renders every element as a rectangular box according to the CSS box model.

The core characteristic of a semantic element is that it clearly communicated its meaning to both the developer and the browser. These elements clearly define its content. Examples are: header, nav, section, article, aside, footer

Align Items vs Align Content
  • align-items effect the alignment of items on the current line.
  • align-content effects the alignment accross lines of a flex-container. This means that this property has no effect on single-line containers.
